dim-template-webapp
=====================
1、安装nodejs环境
=====================
2、安装dim（npm install -g dim）
=====================
3、初始化项目(dim init)
=====================
4、选择spa模式
=====================
5、进入项目根路径，打开命令窗口，执行命令(dim release -cwL)
=====================
6、在当前项目下另开一个命令行窗口，执行命令(dim server start)
=====================
7、在浏览器中打开127.0.0.1:5000，预览项目
=====================
